 # Pythonアプリ ポートフォリオ

## アプリ一覧

### タイピングゲーム（Python GUI）
- ファイル: [`typing.py`](./typing.py)
- 説明: 日本語の文章をタイピングして速度を測定するゲームです。
- 実行方法: Python環境で `python script.py` を実行してください。

### じゃんけんアプリ（Streamlit）
- ファイル: [`janken.py`](./janken.py)
- 説明: ユーザー vs CPUのじゃんけんゲーム
- Webでプレイ：[https://portfolio-renyajanken1.streamlit.app/]

### 電卓アプリ（Streamlit）
- ファイル: [`calculator.py`](./calculator.py)
- 説明: 計算式を入力して答えを出すシンプルな電卓（履歴機能あり）例：（ 1 + 2 ）（ 3 * 2 / 1 ）
- Webでプレイ：[https://portfolio-renyacalculator.streamlit.app/]

### 制作動機と学び
- 実際の業務を想定し、誰にとっても使いやすいアプリを目指しました。
<br> その結果としてユーザー視点での工夫や、スマホでの使いやすさも意識するようになりました。

　- 例：電卓アプリでは、PC上で計算を行う場合に、素早い計算を可能としていましたが、スマホでは「×」「÷」などを使う傾向があり、　&nbsp;PC&nbsp;上と比較して使用しにくいと感じました。
　<br>&nbsp;- スマートフォン向けに制作する際は、GUIの実装（数値や四則演算を直接タップ）を考慮するなど、PC向けとの違いに気付きました。
　<br>&nbsp;今後はスマホ操作にも対応できる設計を目指していきます。

- タイピングゲームでは、文章の自動収集やStreamlit対応も試みましたが、一部実現できなかった点もあり、今後の改善点として残しています。 

- 今回のアプリ制作を通じて、単なる動作の実装にとどまらず、「ユーザーの操作性」や「応用可能性」について深く考えるようになりました。
必要な機能を追加するには、知らない機能や関数が多く、また、実現に必要なコードの理解・発想力等、知識・経験が圧倒的に足りないと実感しました。
<br>今後も継続して学び、より良いアプリ開発を目指します。
